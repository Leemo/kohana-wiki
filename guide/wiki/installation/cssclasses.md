# CSS classes

| Class name      | Description                                                                     |
| --------------- | ------------------------------------------------------------------------------- |
| a.wiki_external | Assigned external links, that contains `://`.                                   |
| a.wiki_empty    | Used for internal links. Assigned to internal links to pages that do not exist. |
| p.note          | Assigned to `[!!] Some note` constructions.                                     |
